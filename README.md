IoT Based Home Automation Using Arduino

Project Description

This project demonstrates a basic home automation system using Arduino Uno. Two appliances (Light and Fan) are controlled using push buttons. LEDs are used to simulate appliances.

Components Used

- Arduino Uno
- 2 Push Buttons
- 2 LEDs
- Resistors
- Jumper Wires

Features

- Light ON/OFF Control
- Fan ON/OFF Control
- Digital Logic Based Switching
- Power Saving Concept
- Wokwi Simulation

Working

Button 1 controls the Light LED and Button 2 controls the Fan LED. Pressing a button toggles the corresponding appliance state.
